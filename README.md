# Portfolio Landing Boilerplate

Boilerplate landing page portofolio menggunakan React, TypeScript, Vite, dan Tailwind CSS.

## Mulai

```bash
npm install
npm run dev
```

## Kustomisasi

- Ubah konten portofolio di `src/data/portfolio.ts`
- Ubah struktur section di `src/App.tsx`
- Ubah style global di `src/index.css`
- Ganti file CV di `public/CV.pdf` atau sesuaikan `resumeUrl`

## Scripts

```bash
npm run dev      # development server
npm run build    # production build
npm run lint     # cek lint
npm run preview  # preview hasil build
```
